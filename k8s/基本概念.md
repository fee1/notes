# Kubernetes
## 简介
```text
    k和s之间有8各字母，简称k8s。
```
## 特性
```text
    1.服务发现与负载均衡: 无需修改应用程序即可使用陌生的服务发现机制。
    2.存储编排: 自动挂载存储系统，包括本地存储。
    3.Secret和配置管理: 部署更新Secret和应用程序的配置时不必重新构建容器镜像，且不必将软件堆栈配置中的秘密信息暴露出来。
    4.批量执行: 除了服务之外，Kubernetes还可以管理你的批处理和CI工作负载，在期望时替换掉失效的容器。
    5.水平扩展: 使用一个简单的命令、一个UI或基于CPU使用情况自动应对程序进行扩缩。
    6.自动化上线和回滚: Kubernetes会分步骤地针对应用或其配置的更改上线，同时监控应用程序运行情况以确保你不会同时终止所有实例。
    7.自动装箱: 根据资源需求和其他约束自动配置，同时避免影响可用性
    8.自我修复: 
```
